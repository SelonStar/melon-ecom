# 项目概览

## 项目名称
仿 Amazon 桌面端电商平台

## 架构
三端分离项目（前端 / 后端 / 其他服务独立部署）

## 页面结构
- 顶部 Header：Logo、搜索栏、账号入口、购物车
- 分类导航条：一级类目横向排列，hover/click 展开二级类目
- 分类商品列表页：点击类目后进入对应商品列表

## 业务逻辑规范

### 下单 & 支付
- 下单和发货逻辑仅做 Demo，不真实扣款或发货
- 用户购买后扣除账户余额
- 下单前必须校验余额是否充足
- 下单完成后展示"支付成功"页面/提示

### 库存
- 核心亮点之一：高并发库存扣减设计
- 需考虑并发安全，防止超卖

## 亮点功能

### AI 试穿
- 核心亮点之一
- 调用现成的第三方 AI 试穿 API 实现
- 非自研模型

## UI 规范
- 页面风格尽量还原 Amazon 官网桌面端
- 当前前端页面较为简陋，后续迭代时优先对齐 Amazon 视觉风格
- 有新的页面/组件需求时，主动参考 Amazon 的布局和交互方式
